# RFSoC-Digital-Transceiver
# RFSoC-Based Digital Transceiver using PYNQ  A complete hardware-software co-design implementation of a digital transmitter and receiver on the AMD/Xilinx Zynq UltraScale+ RFSoC 4x2 platform.  The project demonstrates signal generation, RF transmission, signal reception, FFT analysis, and  (RFDC) configuration using Python and the PYNQ framework.
